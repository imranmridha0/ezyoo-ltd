Wholesale / B2B

Manage B2B portal access and wholesale settings.

B2B Access Code
The B2B access code allows approved stockists to view wholesale pricing on the products page. Share this code only with approved partners.
To change the code, edit the loginB2B() function in wholesale.html.
Wholesale Pricing

Wholesale prices are set per product in the Products editor. When a stockist logs in with the B2B code, they see wholesale prices instead of retail.
