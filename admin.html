Wholesale / B2B

Manage B2B portal access and wholesale settings.

B2B Access Code
The B2B access code allows approved stockists to view wholesale pricing on the products page. Share this code only with approved partners.
To change the code, edit the loginB2B() function in wholesale.html.
Wholesale Pricing

Wholesale prices are set per product in the Products editor. When a stockist logs in with the B2B code, they see wholesale prices instead of retail.

Token Permission Required

Your GitHub token can read but cannot write to this repo. Follow these steps to fix it:

Option A — Classic Token (Recommended)
  1. Go to github.com/settings/tokens
  2. Click Generate new token (classic)
  3. Tick the 'repo' checkbox (full repo access)
  4. Click Generate token and copy it
  5. Click Disconnect in CMS, paste the new token, click Connect
Option B — Download & Replace (Works Right Now)

Download the updated data.json and replace the file in your GitHub repo manually:

Then go to your GitHub repo → data.json → Edit → paste contents → Commit
